## Option 1: From Github

1. Go [here](https://github.com/TheStarloTeam/lunardb)
2. Click "< > Code" and git clone this **__url__**:

```bash
git clone https://github.com/TheStarloTeam/LunarDB.git
```

3. Go to "bin/" and you can find **"lunar.exe"** there.
4. Once find, **SETUP a System Environment Variable** for `bin/`.

### Here's how

1. Search "Environ" into your Windows Search by clicking the Windows button.
2. **Open it**
3. Click **"Environment Variables".**
4. In **System Variables**, scroll until you find **"Path"**.
5. **Click or Open it**
6. And there you can add the location on WHERE the `bin/` for LunarDB is located.

## Option 2: Direct Download

Click [this](https://github.com/TheStarloTeam/LunarDB/raw/refs/heads/main/bin/lunar.exe) to download the executable file of LunarDB.

For the **MSI Setup** click [this](https://github.com/Kazooki123/LunarDB/raw/refs/heads/main/installer/SetupLunar.msi)

### MSI Setup Instructions:

1. Download the MSI from the link above.
2. Run the installer and follow the installation steps.
3. LunarDB will be installed on your machine.

## System Requirements:

- **OS:** Windows (7, 10 & 11), MacOS & Linux
- **RAM:** 2GB-4GB
- **Storage:** 10GB-50GB free space
