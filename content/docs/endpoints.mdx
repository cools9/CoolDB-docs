## Introduction

LunarDB is a powerful database management system that provides a robust API for interacting with your data.
Here is the tutorial on how to set endpoints for your project or app that uses LunarDB.

## Endpoints

The following endpoints are supported by the LunarDB API:

### 1. **GET `/data/{key}`**

Retrieve a value associated with a key from LunarDB.

- **Description:** Fetches the value of the specified key.
- **Example Request:**
  ```bash
  curl -X GET http://localhost:8080/data/myKey
  ```
- **Response:**
  ```json
  {
    "key": "myKey",
    "value": "myValue"
  }
  ```

### 2. **POST /data**

Insert or update a key-value pair in LunarDB.

- **Description:** Adds a new key-value pair or updates an existing one.
- **Request Body:**
  ```json
  {
    "key": "myKey",
    "value": "myValue"
  }
  ```
- **Example Request:**
  ```bash
  curl -X POST http://localhost:8080/data \
  -H "Content-Type: application/json" \
  -d '{"key":"myKey","value":"myValue"}'
  ```
- **Response:**
  ```json
  {
    "message": "Data inserted successfully"
  }
  ```

### 3. **`DELETE /data/{key}`**

Delete a key-value pair from LunarDB.

- **Description:** Removes the specified key-value pair from the database.
- **Example Request:**
  ```bash
  curl -X DELETE http://localhost:8080/data/myKey
  ```
- **Response:**
  ```json
  {
    "message": "Data deleted successfully"
  }
  ```
